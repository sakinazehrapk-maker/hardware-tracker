# Equipment Tracker

Since the biggest issue teenagers face when starting hardware projects is finance so why not maker labs and engineering labs let their students borrow equipment to practice and learn just like a library so i created this web application so the can easily track, check-ins, check-outs and over due equipment.

Built with clean HTML5, CSS3 utilizing the `@layer hack-club-theme` framework, and JavaScript.
Using Hackclubs Css theme <3

---
